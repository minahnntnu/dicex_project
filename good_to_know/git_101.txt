git clone $ADRESS_copied_from_github
- to clone the github folders onto your computer

git status
- used to see the status of you repository, if you are up to date or if you have done changes that needs to be committed
- you need to be in the git folder for this to work

git checkout $BRANCH
- to change the branch you are on
- make sure to never work in the main branch

git add
- to add the files you want to commit

git commit - "you message here"
- to commit to the changes that has been done

git push 
- to send the changes to github
